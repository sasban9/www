2023 AUG

View Cart Drawer

There are four views in Access Cards component:
albums cards sales orders

In orders view, you can see kids per class and order button on each kid. On clicking this button, modal opens with the products of this kid mapped to the photos of the kid in the album. The product description is shown in a flyout popup on the left.
The product can be added using the input box to the bulk cart on the right.
