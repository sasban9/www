HTML

htmlrepresents the root (top-level element) of an HTML document, so it is also referred to as the root element. All other elements must be descendants of this element. There can be only one element in a document.
headThe <head> HTML element contains machine-readable information (metadata) about the document, like its title, scripts, and style sheets. There can be only one element in an HTML document.
titleThe <title> HTML element defines the document's title that is shown in a browser's title bar or a page's tab. It only contains text; tags within the element are ignored.
base|singletonspecifies the base URL to use for all relative URLs in a document. There can be only one <base> element in a document.
meta|singletonThe <meta> HTML element represents metadata that cannot be represented by other HTML meta-related elements, like <base>, <link>, <script>, <style> or <title>.
